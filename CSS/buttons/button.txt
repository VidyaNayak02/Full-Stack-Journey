BUTTON 1 (Like)

.button1 {
    width: 105px;
    height: 70px;
    text-align: center;
    color: white;
    font-size: 30px;
    padding: 20px;
    background-color: red;
    border-radius: 15px;
    margin-left: 15px;
    margin-right: 15px;
}

Explanation:

1. width: 105px;
   - Sets the width of the button to 105 pixels.

2. height: 70px;
   - Sets the height of the button to 70 pixels.

3. text-align: center;
   - Aligns the text horizontally in the center.

4. color: white;
   - Changes the text color to white.

5. font-size: 30px;
   - Increases the size of the text to 30 pixels.

6. padding: 20px;
   - Adds 20 pixels of space inside the button around the text.

7. background-color: red;
   - Gives the button a red background.

8. border-radius: 15px;
   - Rounds the corners of the button.

9. margin-left: 15px;
   - Adds 15 pixels of space to the left of the button.

10. margin-right: 15px;
    - Adds 15 pixels of space to the right of the button.

Hover Effect

.button1:hover {
    cursor: pointer;
    transform: scale(1.05);
    opacity: 0.7;
}

Explanation:

1. cursor: pointer;
   - Changes the mouse cursor into a hand pointer.

2. transform: scale(1.05);
   - Increases the button size by 5% when hovered.

3. opacity: 0.7;
   - Makes the button slightly transparent while hovering.

BUTTON 2 (Share)

.button2 {

    padding: 20px;
    background-color: blue;
    border-radius: 15px;
    margin-left: 15px;
    margin-right: 15px;
}

Explanation:

The properties are the same as Button 1 except:

background-color: blue;
- Sets the background color to blue.

Hover Effect

.button2:hover {
    background-color: lightblue;
    color: black;
    cursor: pointer;
}

Explanation:

1. background-color: lightblue;
   - Changes the background color to light blue.

2. color: black;
   - Changes the text color to black.

3. cursor: pointer;
   - Changes the cursor into a hand pointer.

Unlike Button 1, this button does not grow because transform: scale() is not used.

BUTTON 3 (Subscribe)

.button3 {
    background-color: orange;
    color: white;
    padding: 15px 30px;
    font-size: 20px;
    border: none;
    border-radius: 10px;
    box-shadow: 4px 4px 10px gray;
}

Explanation:

1. background-color: orange;
   - Sets the background color to orange.

2. color: white;
   - Makes the text white.

3. padding: 15px 30px;
   - Adds 15px padding on the top and bottom.
   - Adds 30px padding on the left and right.

4. font-size: 20px;
   - Sets the text size to 20 pixels.

5. border: none;
   - Removes the default border around the button.

6. border-radius: 10px;
   - Makes the corners rounded.

7. box-shadow: 4px 4px 10px gray;
   - Adds a gray shadow behind the button.
   - First 4px: Horizontal shadow distance.
   - Second 4px: Vertical shadow distance.
   - 10px: Blur radius.
   - Gray: Shadow color.

Hover Effect

.button3:hover {
    transform: translateY(-3px);
    box-shadow: 6px 6px 15px gray;
    cursor: pointer;
}

Explanation:

1. transform: translateY(-3px);
   - Moves the button 3 pixels upward when hovered.

2. box-shadow: 6px 6px 15px gray;
   - Makes the shadow larger and blurrier, giving a floating effect.

3. cursor: pointer;
   - Changes the cursor into a hand pointer.

Difference between Padding and Margin

Padding:
- Space inside the element, between the content and the border.
- It increases the space around the content.

Margin:
- Space outside the element.
- It creates distance between one element and another.
